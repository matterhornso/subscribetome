subscribetome · claude code plugin · macOS

Hand Claude Code every API key it needs — and never show it one.

Your keys go to the macOS Keychain. subscribetome injects the real value into a command the instant it runs — you and the model only ever see a placeholder like {{stm:openai:default}}. The secret reaches the shell. It never reaches the chat.

curl api.openai.com -H "Authorization: Bearer {{stm:openai:default}}"

Two ways to switch it on.

Pick the one that fits. Either way it takes under a minute, and you never paste a key.

For humans

Run two commands

Comfortable in a terminal? Run these, then restart Claude Code. Done.

terminal 
claude plugin marketplace add matterhornso/subscribetome
claude plugin install stm@subscribetome

Then: restart Claude Code, run /stm:dashboard, and add your keys.

For Claude Code

Paste one line

New to this? Paste this into Claude Code — it reads the repo, installs the plugin, and activates STM for you.

paste into claude code 
Set up subscribetome for me using https://github.com/matterhornso/subscribetome

Then: restart Claude Code when it tells you to. It never asks you for a key.
