How it works

Keys go in out-of-band. The agent only ever holds a placeholder.

1

Add a key without touching the chat

Enter keys in the localhost dashboard, or import them from your existing .env files. The value goes straight into your macOS Keychain — never through Claude Code.

2

Reference it by placeholder

Each key gets a stable address like {{stm:openai:default}}. That placeholder — not the secret — is all you, the model, and the transcript ever see.

3

The hook injects the real key at runtime

When the model writes a command using a placeholder, subscribetome's PreToolUse hook substitutes the real key the instant the command runs. The shell gets the secret; the conversation never does.

What you get

One inventory for every key, subscription, and dollar.

Runtime key injection

The PreToolUse hook swaps placeholders for real keys in Bash commands — and blocks a placeholder in a file-writing tool.

Two guardrail hooks

UserPromptSubmit blocks a raw key pasted into chat; PostToolUse flags a key that leaked into command output.

Localhost dashboard

A clean web UI to add keys, see every tool and key, and watch your monthly spend — bound to 127.0.0.1, token-authed.

Import from .env

Scan your projects for keys already sitting in .env files, then confirm and pull them into the keychain.

Subscription tracking

Record each tool's plan, monthly cost, and renewal date. See total AI spend at a glance.

OS keychain storage

Secrets live in the macOS Keychain. subscribetome's database holds only an opaque pointer — never a key value.

Security model

Honest about what it guarantees — and what it can't.

What it guarantees
  • ✓Key values live in the macOS Keychain — never in the database, never in the conversation.
  • ✓The model and the transcript only ever hold the placeholder, not the secret.
  • ✓Hooks fail safe: any internal error skips substitution, so a failure can't leak a key.
  • ✓The dashboard binds to loopback only, with a per-run token and a Host/Origin allowlist.
What it can't do
  • ⚠It can't un-leak output — a command that echoes its own key is detected and flagged after the fact, not prevented.
  • ⚠While a command runs, the injected key is briefly visible to local ps. It's kept out of the chat, not the process table.
  • ⚠v1 is macOS only, and revoke is a metadata flag — it doesn't call the provider to rotate the key.
Get started

Install it in two commands.

terminal 
# add the marketplace, then install the plugin
claude plugin marketplace add matterhornso/subscribetome
claude plugin install subscribetome@subscribetome

Requires macOS, Bun, and Claude Code. Restart Claude Code after installing, then run /subscribetome:dashboard to add your first key.
